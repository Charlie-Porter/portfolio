# Vim Cheat Sheet

## Global Commands

- `:h[elp] keyword` - Open help for a keyword
- `:sav[eas] file` - Save file as
- `:clo[se]` - Close current pane
- `:ter[minal]` - Open a terminal window
- `K` - Open man page for the word under the cursor

**Tip:** Run `vimtutor` in a terminal to learn the first Vim commands.

---

## Cursor Movement

- `h` - Move cursor left
- `j` - Move cursor down
- `k` - Move cursor up
- `l` - Move cursor right
- `gj` - Move cursor down (multi-line text)
- `gk` - Move cursor up (multi-line text)
- `H` - Move to the top of the screen
- `M` - Move to the middle of the screen
- `L` - Move to the bottom of the screen
- `w` - Jump forwards to the start of a word
- `W` - Jump forwards to the start of a word (words can contain punctuation)
- `e` - Jump forwards to the end of a word
- `E` - Jump forwards to the end of a word (words can contain punctuation)
- `b` - Jump backwards to the start of a word
- `B` - Jump backwards to the start of a word (words can contain punctuation)
- `%` - Move the cursor to a matching character (e.g., `()` `{}` `[]`)

**Tip:** Prefix a cursor movement command with a number to repeat it. For example, `4j` moves down 4 lines.

---

## Insert Mode (Inserting/Appending Text)

- `i` - Insert before the cursor
- `I` - Insert at the beginning of the line
- `a` - Insert (append) after the cursor
- `A` - Insert (append) at the end of the line
- `o` - Append (open) a new line below the current line
- `O` - Append (open) a new line above the current line

**Escape Insert Mode:**

- `Esc` or `Ctrl + c` - Exit insert mode

---

## Editing

- `r` - Replace a single character
- `R` - Replace multiple characters until `Esc` is pressed
- `J` - Join the line below to the current one with one space in between
- `gJ` - Join the line below to the current one without a space in between
- `cc` - Change (replace) the entire line
- `ciw` - Change (replace) the entire word
- `u` - Undo
- `Ctrl + r` - Redo
- `.` - Repeat the last command

---

## Marking Text (Visual Mode)

- `v` - Start visual mode to mark lines, then issue a command (e.g., yank)
- `V` - Start linewise visual mode
- `Ctrl + v` - Start visual block mode

**Visual Commands:**

- `>` - Shift text right
- `<` - Shift text left
- `y` - Yank (copy) marked text
- `d` - Delete marked text
- `u` - Change marked text to lowercase
- `U` - Change marked text to uppercase

---

## Registers

- `:reg[isters]` - Show register contents
- `"xy` - Yank into register `x`
- `"xp` - Paste from register `x`

**Special Registers:**

- `"` - Unnamed register
- `+` - Clipboard register
- `/` - Last search pattern
- `.` - Last inserted text

---

## Marks and Positions

- `:marks` - List marks
- `ma` - Set a mark `a`
- `` `a `` - Jump to the mark `a`
- `` `" `` - Go to the last edited position

---

## Macros

- `qa` - Record a macro `a`
- `q` - Stop recording
- `@a` - Run macro `a`

---

## Cut and Paste

- `yy` - Yank (copy) a line
- `p` - Put (paste) after the cursor
- `P` - Put (paste) before the cursor
- `dd` - Delete (cut) a line
- `dw` - Delete (cut) the word from the cursor to the start of the next word

---

## Indent Text

- `>>` - Indent line to the right
- `<<` - De-indent line to the left
- `gg=G` - Re-indent the entire file

---

## Exiting

- `:w` - Save the file
- `:q` - Quit
- `:wq` or `:x` - Save and quit
- `:q!` - Quit without saving

---

## Search and Replace

- `/pattern` - Search for `pattern`
- `:%s/old/new/g` - Replace all occurrences of `old` with `new`
- `:%s/old/new/gc` - Replace with confirmation

---

## Tabs

- `:tabnew` - Open a new tab
- `gt` - Move to the next tab
- `gT` - Move to the previous tab
- `:tabc[lose]` - Close the current tab
- `:tabdo command` - Run a command on all tabs

---

## Working with Multiple Files

- `:e file` - Edit a file in a new buffer
- `:bn` - Go to the next buffer
- `:bp` - Go to the previous buffer
- `:ls` or `:buffers` - List all open buffers
- `Ctrl + ws` - Split window
- `Ctrl + ww` - Switch windows
